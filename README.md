# 😷 Face Mask Detection using CNN

A real-time face mask detection system built using a Convolutional Neural Network (CNN) that classifies whether a person is wearing a mask or not.

---

## 🚀 Highlights
- Achieved **97.91% accuracy**
- Real-time detection using webcam
- Image preprocessing and data augmentation
- Face detection using Haar cascades
- End-to-end pipeline (training → detection)

---

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- CNN (Convolutional Neural Network)

---

## 📁 Project Structure
```
face-mask-detection/
│── data_preprocessing.ipynb
│── training.ipynb
│── detection.ipynb
│── README.md
│── requirements.txt
```


---

## ⚙️ How It Works
1. Preprocess dataset images  
2. Train CNN model on masked/unmasked faces  
3. Detect faces using Haar Cascade  
4. Classify mask/no-mask in real-time  

---

## ▶️ How to Run

### 1. Clone the repository
```
git clone https://github.com/Gaurvi-D/Face-Mask-Detection.git
```


### 2. Install dependencies
```
pip install -r requirements.txt
```

### 3. Run the project
- Run `data_preprocessing.ipynb`
- Run `training.ipynb`
- Run `detection.ipynb`
